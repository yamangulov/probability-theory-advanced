# Задача 1. Анализ независимости и коррелированности случайных величин

## Исходные данные
Совместное распределение X (число комнат) и Y (число котов):

| X / Y | 0    | 1    | 2    | 3    |
|---|---|---|---|---|
| 1    | 0,20   | 0,08   | 0,05   | 0,01   |
| 2    | 0,18   | 0,08   | 0,06   | 0,01   |
| 3    | 0,15   | 0,09   | 0,07   | 0,02   |

## 1. Частные распределения случайных величин X и Y

### Частное распределение X (по строкам):

**Формула:** $P(X = x_i) = \sum_{j} P(X = x_i, Y = y_j)$

P(X = 1) = 0,20 + 0,08 + 0,05 + 0,01 = 0,34
P(X = 2) = 0,18 + 0,08 + 0,06 + 0,01 = 0,33
P(X = 3) = 0,15 + 0,09 + 0,07 + 0,02 = 0,33

Проверка: 0,34 + 0,33 + 0,33 = 1,00 ✓

### Частное распределение Y (по столбцам):

**Формула:** $P(Y = y_j) = \sum_{i} P(X = x_i, Y = y_j)$

P(Y = 0) = 0,20 + 0,18 + 0,15 = 0,53
P(Y = 1) = 0,08 + 0,08 + 0,09 = 0,25
P(Y = 2) = 0,05 + 0,06 + 0,07 = 0,18
P(Y = 3) = 0,01 + 0,01 + 0,02 = 0,04

Проверка: 0,53 + 0,25 + 0,18 + 0,04 = 1,00 ✓

## 2. Проверка независимости X и Y

Случайные величины X и Y независимы, если для всех i, j выполняется:
P(X = i, Y = j) = P(X = i) × P(Y = j)

**Формула проверки независимости:** $P(X = x_i, Y = y_j) = P(X = x_i) \times P(Y = y_j)$

Проверим несколько значений:
- P(X = 1, Y = 0) = 0,20
  P(X = 1) × P(Y = 0) = 0,34 × 0,53 = 0,1802
  0,20 ≠ 0,1802 ✗

- P(X = 2, Y = 1) = 0,08
  P(X = 2) × P(Y = 1) = 0,33 × 0,25 = 0,0825
  0,08 ≠ 0,0825 ✗

Поскольку уже первые проверенные значения не равны, **величины X и Y НЕ являются независимыми**.

## 3. Коэффициент корреляции между X и Y

Формула коэффициента корреляции:
ρ(X,Y) = Cov(X,Y) / (σₓ × σᵧ)

где Cov(X,Y) = E[XY] - E[X] × E[Y]

### Найдём E[X]:

**Формула:** $E[X] = \sum_{i} x_i \cdot P(X = x_i)$

E[X] = 1×0,34 + 2×0,33 + 3×0,33 = 0,34 + 0,66 + 0,99 = 1,99

### Найдём E[Y]:

**Формула:** $E[Y] = \sum_{j} y_j \cdot P(Y = y_j)$

E[Y] = 0×0,53 + 1×0,25 + 2×0,18 + 3×0,04 = 0 + 0,25 + 0,36 + 0,12 = 0,73

### Найдём E[XY]:

**Формула:** $E[XY] = \sum_{i} \sum_{j} x_i y_j \cdot P(X = x_i, Y = y_j)$

E[XY] = 1×0×0,20 + 1×1×0,08 + 1×2×0,05 + 1×3×0,01 +
        2×0×0,18 + 2×1×0,08 + 2×2×0,06 + 2×3×0,01 +
        3×0×0,15 + 3×1×0,09 + 3×2×0,07 + 3×3×0,02

E[XY] = 0 + 0,08 + 0,10 + 0,03 + 0 + 0,16 + 0,24 + 0,06 + 0 + 0,27 + 0,42 + 0,18 = 1,54

### Найдём Cov(X,Y):

**Формула:** $Cov(X,Y) = E[XY] - E[X] \cdot E[Y]$

Cov(X,Y) = 1,54 - 1,99 × 0,73 = 1,54 - 1,4527 = 0,0873

### Найдём σₓ² и σᵧ²:

**Формулы дисперсий:** 
$\sigma_X^2 = E[X^2] - (E[X])^2$
$\sigma_Y^2 = E[Y^2] - (E[Y])^2$

**Для X:**
σₓ² = E[X²] - (E[X])²
E[X²] = 1²×0,34 + 2²×0,33 + 3²×0,33 = 0,34 + 1,32 + 2,97 = 4,63
σₓ² = 4,63 - (1,99)² = 4,63 - 3,9601 = 0,6699
σₓ = √0,6699 = 0,8185

**Для Y:**
σᵧ² = E[Y²] - (E[Y])²
E[Y²] = 0²×0,53 + 1²×0,25 + 2²×0,18 + 3²×0,04 = 0 + 0,25 + 0,72 + 0,36 = 1,33
σᵧ² = 1,33 - (0,73)² = 1,33 - 0,5329 = 0,7971
σᵧ = √0,7971 = 0,8928

### Коэффициент корреляции:

**Формула:** $\rho(X,Y) = \frac{Cov(X,Y)}{\sigma_X \cdot \sigma_Y}$

ρ(X,Y) = 0,0873 / (0,8185 × 0,8928) = 0,0873 / 0,7308 = 0,1195

## 4. Коррелированность X и Y

Поскольку коэффициент корреляции ρ(X,Y) = 0,1195 ≠ 0, **величины X и Y являются коррелированными**, но связь между ними слабая.

## 5. Распределение числа котов в двухкомнатной квартире

Условное распределение Y при X = 2:
P(Y = j | X = 2) = P(X = 2, Y = j) / P(X = 2)

P(Y = 0 | X = 2) = 0,18 / 0,33 = 0,5455
P(Y = 1 | X = 2) = 0,08 / 0,33 = 0,2424
P(Y = 2 | X = 2) = 0,06 / 0,33 = 0,1818
P(Y = 3 | X = 2) = 0,01 / 0,33 = 0,0303

Проверка: 0,5455 + 0,2424 + 0,1818 + 0,0303 = 1,0000 ✓

**Среднее число котов в двухкомнатной квартире:**

**Формула:** $E[Y|X=2] = \sum_{j} y_j \cdot P(Y = y_j | X = 2)$

E[Y | X = 2] = 0×0,5455 + 1×0,2424 + 2×0,1818 + 3×0,0303 = 0,6970

## 6. Распределение числа комнат в квартире с одним котом

Условное распределение X при Y = 1:
P(X = i | Y = 1) = P(X = i, Y = 1) / P(Y = 1)

P(X = 1 | Y = 1) = 0,08 / 0,25 = 0,32
P(X = 2 | Y = 1) = 0,08 / 0,25 = 0,32
P(X = 3 | Y = 1) = 0,09 / 0,25 = 0,36

Проверка: 0,32 + 0,32 + 0,36 = 1,00 ✓

**Среднее число комнат в квартире с одним котом:**

**Формула:** $E[X|Y=1] = \sum_{i} x_i \cdot P(X = x_i | Y = 1)$

E[X | Y = 1] = 1×0,32 + 2×0,32 + 3×0,36 = 0,32 + 0,64 + 1,08 = 2,04
