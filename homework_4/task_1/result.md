# Задача 1: Зависимость между полом абитуриентов и выбором специальности

## Решение на бумаге

### Данные
- Мужчины: 355 абитуриентов
  - Точные науки: 197
  - Гуманитарные: 158
- Женщины: 323 абитуриента (102 + 221)
  - Точные науки: 102
  - Гуманитарные: 221

Таблица сопряженности:

| Пол / Специальность | Точные науки | Гуманитарные | Итого |
|---------------------|--------------|--------------|-------|
| Мужчины             | 197          | 158          | 355   |
| Женщины             | 102          | 221          | 323   |
| Итого               | 299          | 379          | 678   |

### Гипотезы
- Основная гипотеза H₀: Пол и выбор специальности независимы.
- Альтернативная гипотеза H₁: Пол и выбор специальности зависимы.

### Статистика критерия
Используем хи-квадрат тест для таблиц сопряженности.

Наблюдаемое значение хи-квадрат:
χ² = Σ [(Oᵢⱼ - Eᵢⱼ)² / Eᵢⱼ]

Eᵢⱼ = (строка итого * столбец итого) / общий итого

E₁₁ = (355 * 299) / 678 = 106545 / 678 ≈ 157.06
E₁₂ = (355 * 379) / 678 = 134545 / 678 ≈ 198.44
E₂₁ = (323 * 299) / 678 = 96677 / 678 ≈ 142.59
E₂₂ = (323 * 379) / 678 = 122617 / 678 ≈ 180.85

χ² = (197 - 157.06)² / 157.06 + (158 - 198.44)² / 198.44 + (102 - 142.59)² / 142.59 + (221 - 180.85)² / 180.85
= (39.94)² / 157.06 + (-40.44)² / 198.44 + (-40.59)² / 142.59 + (40.15)² / 180.85
= 1595.36 / 157.06 + 1635.39 / 198.44 + 1647.25 / 142.59 + 1612.02 / 180.85
≈ 10.16 + 8.24 + 11.55 + 8.91 ≈ 38.86

### Распределение статистики
При H₀ χ² имеет распределение хи-квадрат с df = (число строк - 1) × (число столбцов - 1) = (2-1)×(2-1) = 1 степенью свободы.

### Критическая область
Двусторонняя критическая область (хи-квадрат всегда правосторонний).

Уровень значимости α = 0.05

Критическое значение: χ²_{0.05,1} ≈ 3.841 (из стандартных таблиц распределения хи-квадрат, например, из учебников по статистике или онлайн-ресурсов как NIST или Wolfram Alpha)

Если χ² > 3.841, отвергаем H₀.

### Графическое изображение
На числовой оси χ²:

```
0 ---------- 3.841 ---------- ∞
   [Доверительная]   [Критическая]
                ↑
          Наблюдаемое: 38.86
```

### Вывод
χ² = 38.86 > 3.841, отвергаем H₀. Пол и выбор специальности зависимы.

### Коэффициенты связи
- Коэффициент Крамера V = √(χ² / (n * min(k-1,r-1))) = √(38.86 / (678 * 1)) ≈ √(0.0573) ≈ 0.239 (слабая зависимость)
- Коэффициент Пирсона C = √(χ² / (χ² + n)) = √(38.86 / (38.86 + 678)) ≈ √(38.86 / 716.86) ≈ √(0.0542) ≈ 0.233
