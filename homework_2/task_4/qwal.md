# Определение типа задачи

## Выбранный критерий
**z-критерий для доли или точный биномиальный тест**

## Обоснование выбора
Проверяется корректность нормальной аппроксимации биномиального распределения. При выполнении условий n×p₀ ≥ 5, n×(1-p₀) ≥ 5, n×p̂ ≥ 5, n×(1-p̂) ≥ 5 используется z-критерий. При нарушении условий применяется точный биномиальный тест.
