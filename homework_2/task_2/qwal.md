# Определение типа задачи

## Выбранный критерий
**t-критерий Стьюдента для парных наблюдений**

## Обоснование выбора
Требуется проверка нормальности распределения разностей парных измерений. При нормальном распределении (p ≥ 0.05) используется t-критерий для парных наблюдений. При нарушении нормальности применяется критерий Уилкоксона.
