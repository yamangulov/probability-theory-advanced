# Определение типа задачи

## Выбранный критерий
**t-критерий Стьюдента для одной выборки**

## Обоснование выбора
Поскольку выборка небольшая (n=7) и дисперсия генеральной совокупности неизвестна, требуется проверка нормальности распределения. При нормальном распределении данных (p ≥ 0.05 по критерию Шапиро-Уилка) используется t-критерий Стьюдента. При нарушении нормальности (p < 0.05) применяется непараметрический знаковый тест.
