# Решение задачи 1 "на бумаге"

## Данные задачи
Выборка из 7 батареек: 12,9; 11,6; 13,5; 13,9; 12,1; 11,9; 13,0 В

## 1. Формулировка гипотез

**Основная гипотеза (H₀):** μ = 12 В  
*Математическая запись:* H₀: μ = 12

**Альтернативная гипотеза (H₁):** μ ≠ 12 В  
*Математическая запись:* H₁: μ ≠ 12

## 2. Проверка предпосылок

Поскольку выборка небольшая (n=7) и дисперсия генеральной совокупности неизвестна, необходимо проверить нормальность распределения данных.

**Критерий Шапиро-Уилка:**
- H₀: Данные имеют нормальное распределение
- H₁: Данные не имеют нормальное распределение
- α = 0,05

**Результат:** Статистика W = 0,9462, p = 0,6946  
Поскольку p = 0,6946 > 0,05, нет оснований отвергать гипотезу о нормальности.

**Заключение:** t-критерий Стьюдента применим.

## 3. Вычисление наблюдаемого значения статистики

### Выборочные характеристики:
n = 7
x̄ = (12,9 + 11,6 + 13,5 + 13,9 + 12,1 + 11,9 + 13,0) / 7 = 12,7 В

### Выборочная дисперсия:
s² = Σ(xi - x̄)² / (n-1)
s² = ((-0,1)² + (-1,1)² + (0,8)² + (1,2)² + (-0,6)² + (-0,8)² + (0,3)²) / 6
s² = (0,01 + 1,21 + 0,64 + 1,44 + 0,36 + 0,64 + 0,09) / 6
s² = 4,39 / 6 = 0,7317
s = 0,855 В

### Наблюдаемое значение t-статистики:
tнабл = (x̄ - μ₀) / (s/√n) = (12,7 - 12) / (0,855/√7) = 0,7 / 0,323 = 2,167

## 4. Распределение статистики критерия

При верности H₀ статистика критерия имеет распределение Стьюдента с k = n-1 = 6 степенями свободы:
**t(6)**

## 5. Сторона критической области

Критическая область **двусторонняя**, так как H₁: μ ≠ 12 В.

## 6. Уровень значимости

**α = 0,05**

## 7. Квантили и критические области

Для двусторонней критической области при α = 0,05 и k = 6:

**Левая критическая точка:** t₀.₀₂₅,₆ = **-2,447**
**Правая критическая точка:** t₀.₉₇₅,₆ = **2,447**

**Критические области:**
- (-∞; -2,447] ∪ [2,447; +∞)

**Доверительная область:**
- [-2,447; 2,447]

## 8. Схематический чертеж

```
    |<----- Доверительная область ----->|
----|---------------|--------------------|----
   -2,447         0                   2,447
                 tнабл = 2,167
```

**Положение наблюдаемого значения:** tнабл = 2,167 попадает в доверительную область.

## 9. Статистический вывод

Поскольку |tнабл| = 2,167 < t₀.₉₇₅,₆ = 2,447, наблюдаемое значение статистики критерия **не попадает в критическую область**.

**Ответ:** На уровне значимости 0,05 нет оснований отвергать основную гипотезу H₀. Среднее напряжение батареек можно считать равным 12 В.

**P-значение:** p = 0,074
