# Определение типа задачи

## Выбранный критерий
**z-критерий для разности долей или точный тест Фишера**

## Обоснование выбора
Проверяется корректность нормальной аппроксимации для сравнения долей в двух группах. При выполнении условий n₁×p̂₁ ≥ 5, n₁×(1-p̂₁) ≥ 5, n₂×p̂₂ ≥ 5, n₂×(1-p̂₂) ≥ 5 используется z-критерий. При нарушении условий применяется точный тест Фишера.
