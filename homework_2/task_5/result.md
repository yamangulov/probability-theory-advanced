# Решение задачи 5 "на бумаге"

## Данные задачи
- Группа 1 (принимавшие лекарство): заболели 3 человека, не заболели 172 человека
- Группа 2 (не принимавшие): заболели 32 человека, не заболели 168 человек

n₁ = 3 + 172 = 175 (общий размер группы 1)
n₂ = 32 + 168 = 200 (общий размер группы 2)

p̂₁ = 3/175 = 0,0171 (доля заболевших в группе 1)
p̂₂ = 32/200 = 0,1600 (доля заболевших в группе 2)

## 1. Формулировка гипотез

**Основная гипотеза (H₀):** Лекарство неэффективно (доли заболевших одинаковы)  
*Математическая запись:* H₀: p₁ = p₂  
где p₁, p₂ - истинные доли заболевших в группах 1 и 2

**Альтернативная гипотеза (H₁):** Лекарство эффективно (доля заболевших в группе 1 меньше)  
*Математическая запись:* H₁: p₁ < p₂

## 2. Проверка условий применимости z-теста

Проверим условия применимости z-теста:
- Группа 1: n₁ × p̂₁ = 175 × 0,0171 = 3 (≥ 5) ✗ **НЕ ВЫПОЛНЕНО**
- Группа 1: n₁ × (1-p̂₁) = 175 × 0,9829 = 172 (≥ 5) ✓
- Группа 2: n₂ × p̂₂ = 200 × 0,1600 = 32 (≥ 5) ✓
- Группа 2: n₂ × (1-p̂₂) = 200 × 0,8400 = 168 (≥ 5) ✓

**Заключение:** Условия применимости z-теста НЕ выполнены, так как n₁ × p̂₁ = 3 < 5.  
**Вывод:** Необходимо использовать точный тест Фишера.

## 3. Точный тест Фишера

**Таблица сопряженности:**
```
            Заболели  Не заболели  Всего
Группа 1:       3           172      175
Группа 2:      32           168      200
Всего:        35           340      375
```

**Вычисление отношения шансов:**
OR = (3 × 168) / (32 × 172) = 504 / 5504 = 0,0915

**Наблюдаемое значение статистики Фишера:**
При применении точного теста Фишера получаем:
**P-значение: p = 0,000001**

## 4. Статистический вывод

Поскольку p = 0,000001 < α = 0,05, наблюдаемое значение статистики **попадает в критическую область**.

**Ответ:** На уровне значимости 0,05 отвергаем основную гипотезу H₀. Принимаем альтернативную гипотезу H₁: p₁ < p₂. Лекарство статистически эффективно - доля заболевших среди принимавших лекарство (1,71%) значительно меньше доли заболевших среди не принимавших (16%).

## 5. Практическая интерпретация

Результат демонстрирует высокую эффективность лекарства:
- Снижение заболеваемости с 16% до 1,71%
- Относительное снижение риска: (16% - 1,71%)/16% = 89,3%
- Абсолютное снижение риска: 16% - 1,71% = 14,29%
- Отношение шансов: 0,0915 (снижение в 10,9 раза)
