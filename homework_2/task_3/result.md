# Решение задачи 3 "на бумаге"

## Данные задачи
Выборка 1 (горожане): 132, 111, 119, 138, 200, 131, 138, 170, 159, 140
Выборка 2 (селяне): 115, 190, 127, 155, 148, 121, 116, 121, 197

n₁ = 10, n₂ = 9

## 1. Формулировка гипотез

**Основная гипотеза (H₀):** Горожане и селяне имеют одинаковое давление  
*Математическая запись:* H₀: M₁ = M₂

**Альтернативная гипотеза (H₁):** Горожане имеют более высокое давление, чем селяне  
*Математическая запись:* H₁: M₁ > M₂

## 2. Проверка предпосылок

Для независимых выборок небольшого размера необходимо проверить нормальность распределения в обеих группах.

**Критерий Шапиро-Уилка для горожан:**
- H₀: Данные имеют нормальное распределение
- H₁: Данные не имеют нормальное распределение
- α = 0,05
- **Результат:** W = 0,9101, p = 0,281

**Критерий Шапиро-Уилка для селян:**
- H₀: Данные имеют нормальное распределение
- H₁: Данные не имеют нормальное распределение
- α = 0,05
- **Результат:** W = 0,8287, p = 0,043

**Заключение:** Для горожан p = 0,281 > 0,05 (нормальное), для селян p = 0,043 < 0,05 (не нормальное). Поскольку хотя бы одна выборка не имеет нормального распределения, t-критерий Стьюдента неприменим. Используется непараметрический критерий Манна-Уитни.

## 3. Критерий Манна-Уитни

**Суть метода:** Критерий Манна-Уитни ранжирует все наблюдения из обеих выборок, а затем суммирует ранги каждой группы и сравнивает эти суммы.

### Выборочные характеристики:
**Горожане (выборка 1):**
x̄₁ = (132 + 111 + 119 + 138 + 200 + 131 + 138 + 170 + 159 + 140) / 10 = 143,8
Медиана₁ = (138 + 140) / 2 = 139

**Селяне (выборка 2):**
x̄₂ = (115 + 190 + 127 + 155 + 148 + 121 + 116 + 121 + 197) / 9 = 143,3
Медиана₂ = 148

### Статистика критерия Манна-Уитни:
При применении критерия Манна-Уитни получаем:
**Статистика:** U = 50
**P-значение:** p = 0,357

## 4. Статистический вывод

Поскольку p = 0,357 > α = 0,05, наблюдаемое значение статистики **не попадает в критическую область**.

**Ответ:** На уровне значимости 0,05 нет оснований отвергать основную гипотезу H₀. Нельзя считать верной гипотезу о том, что жители городов имеют в среднем более высокое давление, чем жители сельской местности.

**Практическая интерпретация:** Поскольку данные не имеют нормального распределения, использован непараметрический критерий Манна-Уитни. Этот тест сравнивает медианы, а не средние значения.
