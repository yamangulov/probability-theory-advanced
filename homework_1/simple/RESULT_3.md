# Результаты выполнения задания 3

Строим точный доверительный интервал для среднего μ с надёжностью 0.95.

Дано: генеральная совокупность распределена нормально со средним квадратическим отклонением σ=5, выборочная средняя x̄=24.15, объём выборки n=100.

## Шаг 1: Выбрать подходящую формулу доверительного интервала

Поскольку дисперсия генеральной совокупности известна (σ=5), используем точный доверительный интервал для среднего при известной дисперсии:

$$P \left( \bar{x} - z_{1-\frac{\alpha}{2}} \cdot \frac{\sigma}{\sqrt{n}} \leq \mu \leq \bar{x} + z_{1-\frac{\alpha}{2}} \cdot \frac{\sigma}{\sqrt{n}} \right) = 1-\alpha$$

Где α=0.05, 1-α=0.95.

## Шаг 2: Применить формулу для доверительного интервала

Подставим известные значения:

α = 0.05

1-α = 0.95

z_{1-α/2} = z_{0.975} ≈ 1.96

σ = 5

x̄ = 24.15

n = 100

Сначала вычислим стандартную ошибку среднего:

$$\frac{\sigma}{\sqrt{n}} = \frac{5}{\sqrt{100}} = \frac{5}{10} = 0.5$$

Теперь вычислим границы интервала:

Нижняя граница:

$$\bar{x} - z_{1-\frac{\alpha}{2}} \cdot \frac{\sigma}{\sqrt{n}} = 24.15 - 1.96 \cdot 0.5 = 24.15 - 0.98 = 23.17$$

Верхняя граница:

$$\bar{x} + z_{1-\frac{\alpha}{2}} \cdot \frac{\sigma}{\sqrt{n}} = 24.15 + 1.96 \cdot 0.5 = 24.15 + 0.98 = 25.13$$

## Шаг 3: Получить конкретный доверительный интервал для среднего

Доверительный интервал с надёжностью 0.95:

$$23.17 \leq \mu \leq 25.13$$

Или в интервальной форме: (23.17, 25.13)
